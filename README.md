# WordDictionarySearch

Tool for querying word dictionary  
## five functions:
	1.Definition(单词定义)  
	2.WordFamily(形近词)  
	3.Synonym(同义词)  
	4.Antonym(反义词)  
	5.ContextualSentence(例句)  
    6.ALL

You can input word in 'Word.txt' like as :  
  
achieve  
acquisition  
administration  
affect  
appropriate  
aspects  
assistance  
  
Then run SearchWord.py
