# WordDictionarySearch

Tool for querying word dictionary
five functions:
	1.Definition(单词定义)\n
	2.WordFamily(形近词)\n
	3.Synonym(同义词)\n
	4.Antonym(反义词)\n
	5.ContextualSentence(例句)\n

You can input word in 'Word.txt' like as :

achieve
acquisition
administration
affect
appropriate
aspects
assistance

Then run SearchWord.py
